# SortManager

Sort Manager is a backend Java program that allows users to compare the speeds of six different sorting algorithms. The six algorithms used were:

1. Binary Tree Sort
2. Bubble Sort
3. Insertion Sort
4. Merge Sort
5. Quick Sort
6. Selection Sort
